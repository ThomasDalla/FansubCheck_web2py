Fansub Check

Pour un Fansub de qualité !

Le principe de FansubCheck est simple : Vous envoyez votre fichier de sous-titre (format SSA ou ASS) et le site génère un script sur lequel vous pouvez effectuer des commentaires sur chaque sous-titres.

Vous pouvez choisir que votre QC (Quality Check) soit public ou non listé.
Un QC public sera visible par tous les membres de FansubCheck qui pourront alors effectuer leurs commentaires.
Un QC non listé ne sera accessible que par ceux à qui vous donnaient le lien (par exemple sur le forum de votre team).

/!\ ATTENTION /!\ En beta, les données sont régulièrement supprimées...
